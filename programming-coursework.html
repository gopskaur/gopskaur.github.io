We Do Programming Coursework

We Do Programming Coursework provides programming coursework to be professionally explained and completed for collage and university students ranging on all levels of expertise.

This project involved creating a logo and colour palette to define assets for the design of future materials. Four infographic banners were also created to illustrate the main aspects to the service that We Do Programming Coursework offers.

The tools used to create all designs within this project involved using Adobe Illustrator.

After branding Programming Courework, I moved onto designing some banners to be placed on their website.

Whilst creating the illustrations I thought about the best ways users will understand what the company does and who they vision the company to be. These cognitive points were fully considered whilst creating the banners, so that users would be able to understand and relate to Programming Coursework's service.
