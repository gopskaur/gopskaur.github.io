John Lewis: Space Data Capture

Space Data Capture is an iOS app which allows John Lewis branch employees to help maintain product capacity information for replenishment.

Over the duration of this project, I have created some variations of the logo design, icons and wireframe layouts to justify the content and layout for the app. Continuing throughout the production, I will work closely with Sketch to mock-up the visual designs for each individual screen.

All of the wireframe designs will then be appointed to the software developers, so that all the changes can be incorporated. My contribution to the project will flow onto working aside the developers to make smart usability improvements in light of getting the MVP out for usability testing.

Whilst creating variations of the logo design, I thought about ideas in relation to the purpose of the project. How would the logo best present the functionality of the app? and What would make the users want to click on the app's logo? came into mind. As soon as some mindful and web research was done, I delved into Sketch to shape up all elements and bind them together to create each of the logo designs.

In order to break down information within the app, a set of icons were created to help employees visualise the types of fixtures where products would be replenished.

The selection of wireframe designs produced demonstrate how various content and user interface elements is to be displayed on screens.

Whilst thinking about the creating a viable outcome for the app, I worked with all of the Space Data project team members to discuss ideas for aiming towards the best user experience.
